## Overview

Logseq Professional Suite is a complete networked outlining platform that turns a habit of daily journaling into a queryable, interconnected knowledge base stored entirely as local markdown files.

Queries written directly in notes pull together blocks matching specific criteria, turning the journal into a dynamic database that surfaces patterns across months or years of entries.

A plugin ecosystem extends Logseq with community-built tools for task management, spaced repetition, and visualization, adapting the core outliner to a wide range of personal knowledge management approaches.

Version control compatibility through Git allows an entire knowledge base to be tracked, branched, and synced using standard developer tooling rather than a proprietary sync service.

Logseq Professional Suite is the platform for users who want their knowledge base to grow organically from daily writing, built on open, local data they fully control.

## Performance & Integration

Logseq is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Logseq processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
